# 主题切换功能指南

## 🎨 主题切换位置

主题切换按钮位于以下位置：

### 1. 侧边栏顶部（移动端可见）
- 在侧边栏的右上角，紧邻关闭按钮
- 在移动设备上更容易访问

### 2. 侧边栏底部（桌面端可见）
- 在侧边栏底部的footer区域
- GitHub链接旁边

### 3. 头部导航栏
- 在页面头部的右上角
- 模型选择器旁边

## 🌈 主题模式

### 1. 亮色主题 🌞
- 适合日间使用
- 明亮的背景，深色文字
- 减少眼睛疲劳

### 2. 暗色主题 🌙
- 适合夜间使用
- 深色背景，浅色文字
- 保护视力

### 3. 跟随系统 🖥️
- 自动适配系统主题设置
- 跟随操作系统的明暗模式
- 智能切换

## 🔄 切换逻辑

点击主题按钮会按以下顺序循环切换：
```
亮色 → 暗色 → 跟随系统 → 亮色
```

## 💾 设置保存

- 主题选择会自动保存到浏览器的 localStorage
- 下次访问时会自动恢复上次的设置
- 跟随系统模式下会实时响应系统主题变化

## 🔧 技术实现

- 使用 React Context 管理主题状态
- CSS 变量实现主题令牌系统
- Tailwind CSS 的 `dark:` 前缀自动应用暗色样式

## 🐛 故障排除

如果看不到主题切换按钮：

1. **刷新页面**：确保加载最新代码
2. **检查控制台**：查看是否有JavaScript错误
3. **清除缓存**：强制刷新 (Ctrl+F5 / Cmd+Shift+R)
4. **检查侧边栏**：确保侧边栏已打开

## 📱 移动端使用

- 点击左上角的菜单按钮打开侧边栏
- 主题切换按钮在侧边栏顶部
- 选择主题后侧边栏会自动关闭
